# iPhoneme: Gesture-based Proxy Evaluation Prototype

Welcome to the **iPhoneme Proxy Experiment**. 

This repository contains an **in-browser proxy evaluation prototype** for an ALS-oriented Gaze + Phoneme interface. You do not need to install any software or backend servers to try it out. 

## 🚀 Run the Experiment
You can run the experiment completely in your browser without any installation:

👉 **[Launch iPhoneme Experiment](https://ChachaTJ.github.io/Gaze_trigger_ex/)**

*Note: The prototype requires Webcam access for head tracking via MediaPipe.*

## Local Development
If you want to run the code locally, just start any static HTTP server in the root directory:
```bash
python3 -m http.server 8000
```
Then visit `http://localhost:8000/`.
